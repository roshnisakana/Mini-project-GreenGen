# 🌱 GreenGen

## Overview

GreenGen is a web-based sustainability platform designed to promote environmental awareness and encourage eco-friendly practices. The application provides users with tools and information related to waste management, recycling, and sustainable living through an interactive and user-friendly interface.

## Features

* User-friendly web interface
* Sustainable waste management solutions
* Environmental awareness resources
* Database-driven data management
* Responsive and interactive design

## Technologies Used

* Python
* Flask
* HTML5
* CSS3
* JavaScript
* MySQL

## Project Structure

* Frontend: HTML, CSS, JavaScript
* Backend: Python Flask
* Database: MySQL

## Objective

To leverage technology in promoting environmental sustainability and creating awareness about responsible waste management and green living practices.
